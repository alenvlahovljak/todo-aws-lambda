## Todo list with AWS Lambda

## technologies
* NodeJS,
* Express.js,
* ejs,
* MongoDB

## preparation
* NODE_ENV=staging
* PORT=4000
* MONGODB_URI={uri}
* LAMBDA_API={api}
* REGION={region}

## instalation
```node.js
  git clone https://github.com/alenvlahovljak/todo-aws-lambda.git
  run npm instal
  run mongod in the terminal
  run npm dev
  open localhost:4000
  ```
## requirements
  * NodeJS
  * MongoDB (or AtlasDB)
